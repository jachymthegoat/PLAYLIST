# PLAYLIST
Ukol z PPA
Program pro generování playlistu nejlepších skladeb. Každá skladba o sobě uchovává informace o svém názvu, délce a uživatelském hodnocení. Kompletní seznam skladeb je v playlistu, ze kterého je možné nechat vygenerovat nový playlist, na kterém jsou náhodně uspořádané skladby s nejlepším hodnocením a jejich celková délka bude přibližně odpovídat zadané délce. Pokud není dostatek skladeb, program o tom uživatele informuje. Pro řazení je zde použitý bubble sort.
